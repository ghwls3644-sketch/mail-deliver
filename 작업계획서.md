## 실제 잔여 작업 (`restart()` 정리)

`main.py`의 `restart()` 함수 내 불필요 코드 3줄:

```python
import importlib, game.mail as gm
importlib.reload(gm)           # os.execv 이후 의미 없음
application.base.graphicsEngine.renderFrame()  # 빈 씬 렌더 → 검은 화면 플래시
python_object = application    # 죽은 코드
```

→ `os.execv` 한 줄만 남기고 제거하면 됨. 낮은 우선순위.
