# 에디터 클래스 구조 스케치

> editor.py 클래스화 작업 기록
> 리팩토링 일자: 2026-04-01

---

## 클래스 개요

```
MapEditor(Entity)
```

Ursina의 `Entity`를 상속해서 `input(self, key)`와 `update(self)`가 엔진에 자동 연결됨.

---

## 클래스 변수 (공유 상수)

| 변수명 | 값 | 설명 |
|--------|----|------|
| `CAM_PAN_SPEED` | 25 | WASD 이동 속도 |
| `CAM_ZOOM_STEP` | 5 | 스크롤 줌 단계 |
| `CAM_ROT_SPEED` | 80 | Ctrl+마우스 회전 감도 |
| `HELP_LINES` | 리스트 | 단축키 패널 텍스트 목록 |

---

## 인스턴스 변수 (self.xxx)

### 에디터 상태
| 변수명 | 초기값 | 설명 |
|--------|--------|------|
| `cat_idx` | 0 | 현재 선택된 카테고리 인덱스 |
| `model_idx` | 0 | 현재 선택된 모델 인덱스 |
| `rot_y` | 0 | 오브젝트 배치 회전값 (도) |
| `placed` | [] | 배치된 오브젝트 목록 `[(Entity, dict), ...]` |
| `current_map` | None | 현재 작업 중인 맵 이름 (확장자 제외) |
| `dialog_open` | False | 다이얼로그 열림 여부 (입력/카메라 차단) |

### UI 참조
| 변수명 | 설명 |
|--------|------|
| `info_txt` | 상단 정보 텍스트 |
| `status_txt` | 하단 상태 메시지 텍스트 |
| `help_panel` | 우측 단축키 패널 (H 키 토글) |
| `grid_root` | 그리드 Entity (G 키 토글) |
| `indicator` | 배치 위치 인디케이터 |

### 미리보기
| 변수명 | 설명 |
|--------|------|
| `prev_buf` | 오프스크린 렌더 버퍼 |
| `prev_tex` | 미리보기 텍스처 |
| `ps` | 미리보기 전용 씬 NodePath |
| `prev_wrapper` | 현재 미리보기 모델 NodePath |
| `prev_rot_y` | 미리보기 자동 회전 각도 |

### 다이얼로그
| 변수명 | 설명 |
|--------|------|
| `save_dialog` | 저장 다이얼로그 Entity |
| `save_field` | 저장 이름 입력 필드 |
| `load_dialog` | 불러오기 다이얼로그 Entity |

### UI 색상
| 변수명 | 설명 |
|--------|------|
| `C_BG` | 하늘색 반투명 패널 배경 |
| `C_BORDER` | 진한 파랑 테두리 |
| `C_TITLE` | 제목 텍스트 색 |
| `C_KEY` | 단축키 텍스트 색 |
| `C_DESC` | 설명 텍스트 색 |
| `C_INFO` | 상단 정보 텍스트 색 |

## 메서드 목록

### 초기화
| 메서드 | 설명 |
|--------|------|
| `__init__` | 색상/상태 변수 초기화 → 3개 setup 호출 |
| `_setup_scene()` | 바닥, 그리드, 인디케이터, 카메라 초기 배치 |
| `_setup_preview_buffer()` | 오프스크린 버퍼 및 미리보기 씬/카메라 생성 |
| `_setup_ui()` | 상단 정보바, 단축키 패널, 미리보기 패널, 상태바 생성 |

### 헬퍼
| 메서드 | 설명 |
|--------|------|
| `_panel(parent, w, h, cx, cy)` | 반투명 배경 + 테두리 패널 생성 |
| `_cur_models()` | 현재 카테고리의 모델 목록 반환 |
| `_cur_item()` | 현재 선택된 (이름, 경로, skey) 반환 |
| `_make_checker_texture()` | @staticmethod, 체커보드 텍스처 PNG 생성 |

### UI 갱신
| 메서드 | 설명 |
|--------|------|
| `_refresh_ui()` | 상단 정보 텍스트 갱신 + 미리보기 갱신 |
| `_show_status(msg, col)` | 하단 상태 메시지 표시 (2.5초 후 자동 소멸) |
| `_update_preview()` | 현재 모델을 미리보기 씬에 로드 및 자동 스케일 |

### 배치 / 삭제
| 메서드 | 설명 |
|--------|------|
| `_place()` | 마우스 위치에 현재 모델 그리드 스냅 배치 |
| `_delete_nearest()` | 마우스 근처(3유닛 이내) 오브젝트 삭제 |
| `_undo()` | 마지막 배치 취소 |

### 저장 / 불러오기
| 메서드 | 설명 |
|--------|------|
| `_do_save(name)` | `game/maps/name.json` 으로 실제 저장 |
| `_do_load(name)` | `game/maps/name.json` 불러와 씬 재구성 |
| `_open_save_dialog()` | 이름 입력 다이얼로그 표시 |
| `_close_save_dialog()` | 저장 다이얼로그 닫기 |
| `_confirm_save()` | 입력 이름 검증 후 `_do_save` 호출 |
| `_open_load_dialog()` | 맵 목록 다이얼로그 표시 |
| `_close_load_dialog()` | 불러오기 다이얼로그 닫기 |
| `_select_map(name)` | 다이얼로그 닫고 `_do_load` 호출 |

### 엔진 루프
| 메서드 | 설명 |
|--------|------|
| `input(self, key)` | 키 입력 처리 (다이얼로그 중엔 Esc/Enter만) |
| `update(self)` | 매 프레임: 카메라 이동/회전, 미리보기 회전, 인디케이터 |

## 실행 흐름

```
app = Ursina(...)
Text.default_font = ...
render.setShaderAuto()
camera.background_color = ...

editor = MapEditor()   ← __init__ 에서 씬/UI 전부 구성
app.run()
```

## 파일 저장 구조

```
game/
  maps/
    맵이름.json   ← Ctrl+S 로 저장, L 로 불러오기
```

### JSON 오브젝트 형식
```json
{
  "objects": [
    {
      "name": "straight",
      "model": "assets/.../road-straight.glb",
      "pos": [2.0, 0, 4.0],
      "scale": [2, 2, 2],
      "rot_y": 90
    }
  ]
}
```
